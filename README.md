> 开发模式预览
- 安装nodejs环境（nodejs>= 6.0）
- 安装依赖 npm install
- 运行代码 npm start

> 打包
- npm run build （生成在build文件夹下）

## serve文件夹下部分后端 
- npm install
- npm start

## 目录结构
- src/ 前端源码
    - actions/ redux的action部分
    - components/ 组件
    - containers/ 容器
    - reducers/ redux的reducer部分
    - static/   静态文件
    - store/ reudx的store部分
    - const.js 常量配置
    - index.js 入口文件
- serve/ nodejs后端部分代码
